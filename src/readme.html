MegaMap 0.9 Readme

The MegaMap distribution consists of these files:
  • readme.html (this file)
  • MegaMap-0.9.jar
  • commons-collections-3.1.jar
  • commons-logging-1.0.3.jar
To use MegaMap, make sure that all of the above .jar files are in your classpath. See http://MegaMap.sf.net for instructions on how to use MegaMap with your software.
